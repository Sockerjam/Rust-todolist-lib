# To Do List

A to-do list core written in **Rust**, with **iOS** bindings generated by **UniFFI**.

**iOS demo** available in **ToDoList_iOS**

**Rust** core logic code located in: **core_logic**.
**UniFFI** executable located in: **uniffi-bindgen**
